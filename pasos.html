Pasos

Ruby on Rails, comúnmente conocido como Rails,

Paso 1: Instalar Ruby

Primero, necesitas instalar Ruby. Puedes usar un gestor de versiones como rbenv o rvm. Aquí te muestro cómo hacerlo con rbenv:

1.- Instalar rbenv y ruby-build:

git clone https://github.com/rbenv/rbenv.git ~/.rbenv

echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc

echo 'eval "$(rbenv init -)"' >> ~/.bashrc

exec $SHELL

git clone https://github.com/rbenv/ruby-build.git

~/.rbenv/plugins/ruby-build

2.- Instalar Ruby:

rbenv install 3.2.0

rbenv global 3.2.0

Paso 2: Instalar Bundler

Bundler es una herramienta para gestionar las gemas de Ruby. Instálalo con el siguiente comando:

gem install bundler

Paso 4: Crear una nueva aplicación Rails

Una vez que Rails esté instalado, puedes crear una nueva aplicación con:

rails new nombre_de_tu_aplicacion

Paso 5: Iniciar el servidor

Navega al directorio de tu nueva aplicación y arranca el servidor:

cd nombre_de_tu_aplicacion rails server
